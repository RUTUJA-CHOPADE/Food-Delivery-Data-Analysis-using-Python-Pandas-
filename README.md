# Food Delivery Data Analysis

## Project Overview
This project analyzes a food delivery dataset by merging orders, users, and restaurant information to generate actionable insights. The analysis focuses on:
- City-wise and membership-wise revenue
- Cuisine performance
- Restaurant ratings vs revenue
- User behavior and order trends

## Files in this Repository
1. `orders.csv` - Raw order data
2. `users.json` - User information including membership and city
3. `restaurants.csv` - Restaurant data with cuisine type and ratings
4. `final_food_delivery_dataset.csv` - Consolidated dataset (optional)
5. `hackathon.ipynb` - Jupyter Notebook containing all code, analysis, and answers to questions
